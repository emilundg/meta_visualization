Design choices


A description button exists for the user to get more information about what the different categories (e.g Thefts) mean. Press it!

Below a description of the different breaches exist

Theft is the act of physically taking someone elses property without their permission.

Unauthorized Access is when someone gains access to access to an area of a system that they are not permitted to access

Loss is when information is destroyed by different type of failures in a system.

Hacking/IT Incidents is when someone exploits a system by using different vulnerabilities such as bugs or exploits to access computer systems.

Disposal includes incident where data has been overwritten and/or destroyed.

Other incidents include data breaches that could not be categorized by the other ones.

Highlight!


Group presentation provided feedback about not having information of what the different categories mean. Therefor it was added to the individual project!
